# Java OOP Pratikleri ve Nesne Tabanlı Programlama Notları

Bu depo, Nesne Tabanlı Programlama (OOP) kavramlarını pekiştirmek amacıyla geliştirilmiş temel Java pratiklerini içermektedir.

## İçerik ve Paket Yapısı

Proje içerisinde yer alan paketler ve ele aldıkları konular şunlardır:

* **`Degisken/`**: Java'da ilkel (primitive) veri tiplerinin ve sınıflara ait üye değişkenlerin (instance variables) varsayılan değerlerinin incelenmesi.
* **`Hesapla/`**: Nesnelerin metotlara referans olarak gönderilmesi (Pass-by-Reference mantığı) ve metot içinde nesne durumunun değiştirilmesi.
* **`OgrenciDers/`**: Sınıflar (Öğrenci ve Ders) arasındaki bire-bir ve bire-çok ilişkilerin (Association) modellenmesi.
* **`Ogrenci/`**: Sınıflar arası daha karmaşık yapıların (Öğrenci, Ders, Bölüm) diziler kullanılarak çoka-çok ilişkilerle (Aggregation) birbirine bağlanması.

## Kullanılan Teknolojiler
* Java (JDK)
* Nesne Yönelimli Tasarım (OOP) Prensipleri
